px-webapp-webdav
================

The Px Webapp WebDAV extension.

This module is based on WebDAV implementation from
http://sourceforge.net/projects/webdav-servlet

It will be soon deprecated, since the SF project is not advancing really fast, and AH provides (in SVN) a 100% WebDAV implementation within it's Danube server:

WWW:
http://www.abstracthorizon.org/danube

Trac:
http://trac.abstracthorizon.org/danube

SVN:
http://svn.abstracthorizon.org/danube/trunk/
